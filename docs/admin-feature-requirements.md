# 管理员功能需求分析

本文档是针对校园问卷投票系统 **管理员功能** 的增量需求分析，需与 `requirements-analysis.md` 配合阅读。

## 1. 功能目标

在现有普通用户功能基础上，新增管理员角色。管理员负责审批用户发布的问卷，并有权限结束和删除任意问卷。

## 2. 使用者与权限（更新）

系统用户角色从一种扩展为两种：

| 角色 | 说明 |
|---|---|
| 普通用户（user） | 默认角色，注册后即为普通用户 |
| 管理员（admin） | 拥有审批、结束、删除任意问卷的权限 |

### 2.1 普通用户权限（更新）

在原有权基础上，新增以下约束：

- 发布问卷后，问卷状态为"待审批"，**不会立即出现在问卷列表中**。
- 只能对自己发布的、且状态为"已结束"的问卷执行结束操作。
- 删除权限不变：只能删除自己发布的问卷。

### 2.2 管理员权限

管理员登录后可以：

- 查看所有待审批的问卷，并执行审批通过操作。
- 查看所有问卷（包括待审批、已通过、已结束）。
- **结束任意已通过的问卷**（投票停止后，用户不可再投票，但仍可查看结果）。
- **删除任意问卷**（无论是否为自己发布）。
- 管理员自己也可以像普通用户一样发布问卷和投票。

### 2.3 问卷状态说明

问卷在生命周期中有三种状态：

```
用户发布问卷 → [待审批] → 管理员审批 → [已通过] → 投票进行中
                                               → 管理员/发布者结束 → [已结束]
```

| 状态 | 含义 | 可见性 |
|---|---|---|
| 待审批（pending） | 用户已发布，等待管理员审批 | 仅管理员可见 |
| 已通过（approved） | 管理员已审批通过，用户可以投票 | 所有登录用户可见 |
| 已结束（ended） | 投票已关闭，不可再投票，结果仍可查看 | 所有登录用户可见 |

## 3. 核心页面需求（新增与变更）

### 3.1 问卷列表页面（变更：vote_list.jsp）

变更内容：

- 普通用户只看到"已通过"和"已结束"的问卷，看不到"待审批"问卷。
- 已结束的问卷不再显示"去投票"按钮，改为显示"已结束"标记。
- 管理员看到所有问卷，并在待审批问卷旁显示"审批通过"按钮。
- 管理员在任意已通过问卷旁显示"结束"按钮。
- 管理员在任意问卷旁显示"删除"按钮。

### 3.2 问卷发布页面（变更：vote_publish.jsp）

变更内容：

- 发布成功后的提示信息改为："问卷已提交，等待管理员审批后即可显示在列表中。"

## 4. 管理员账号管理

当前阶段不开发管理员创建界面。管理员账号通过以下方式创建：

- 直接通过 SQL 在数据库中插入管理员账号，或将现有用户的 `role` 字段修改为 `admin`。

后续如有需要，可在管理员面板中开发用户管理功能。

## 5. 业务流程（新增与变更）

### 5.1 发布问卷流程（变更）

1. 用户发布问卷。
2. 问卷状态设置为"待审批"。
3. 提示用户等待管理员审批。
4. 管理员登录后看到待审批问卷。
5. 管理员点击"审批通过"。
6. 问卷状态变为"已通过"，出现在所有用户的问卷列表中。

### 5.2 审批流程（新增）

1. 管理员登录后进入问卷列表。
2. 在待审批区域看到所有待审批问卷。
3. 点击"审批通过"按钮。
4. 问卷状态变为"已通过"。
5. 刷新列表，该问卷出现在已通过区域。

### 5.3 结束问卷流程（新增）

1. 管理员或问卷发布者点击"结束"按钮。
2. 系统校验身份：当前用户是管理员或问卷发布者。
3. 问卷状态变为"已结束"。
4. 该问卷不再允许投票，但结果仍可查看。

### 5.4 删除问卷流程（变更）

原流程：只有发布者本人可以删除。

新流程：

1. 管理员或问卷发布者点击"删除"按钮。
2. 进入删除确认页面。
3. 系统校验身份：当前用户是管理员或问卷发布者。
4. 确认后删除问卷。

## 6. 校验规则（新增与变更）

- 审批：必须是管理员，问卷状态必须为"待审批"。
- 结束：必须是管理员或发布者本人，问卷状态必须为"已通过"。
- 删除：必须是管理员或发布者本人。
- 普通用户不能看到待审批问卷。
- 不能对已结束的问卷进行投票。

## 7. 验收标准

在原有验收标准基础上，新增：

- 管理员账号可以正常登录。
- 普通用户发布问卷后，问卷不会立即显示在列表中。
- 管理员可以看到待审批问卷并审批通过。
- 审批通过后，所有用户可以在列表中看到该问卷并投票。
- 管理员可以结束任意已通过的问卷。
- 发布者可以结束自己发布的已通过问卷。
- 已结束的问卷不允许投票。
- 管理员可以删除任意问卷。
- 发布者可以删除自己的问卷。
- 非管理员、非发布者不能删除别人的问卷。
